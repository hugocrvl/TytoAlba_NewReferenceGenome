# Allele_Sorting

1. The first step was to modified jellyfish v2.3.0 (https://github.com/gmarcais/Jellyfish/releases/tag/v2.3.0) 
   to add the possibility to create jellyfish bin files that contain kmer present only in one library and are absent from a list of other libraries.
   Differences with the originel source code are provided in the file **Jellyfish-2.3.0-patch.txt**
2. Then we collected 49nt kmers present in balanced random selections (generated by script **collectKmerWithoutN.R**) of illumina reads of each parent using the script **collect.sh** (which uses **collectKmerWithoutN.pl**)
3. Then we created libraries of 49nt kmers present in illumina libraries from one parent but absent in the other parent using the script **prepareDB.sh**
4. Then we counted occurrences of these parental specific kmers on each Oxford Nanopore read using the script **predict_ON_ReadParent25k.sh**
5. Then we predicted the parental origin of each read using the script **predict_ON_ReadParent25k.R**
